Ensure you have the latest "up-to-date" information regarding wordlists. Default or otherwise.

War files need to be unzipped when they are deployed, and you need to know where to go. If using msfvenom, make sure to:
jar -xvf your_war.war

and make note of the shell.jsp page (msfvenom makes it a random generated file name). Navigate there once your war is deployed and started.
