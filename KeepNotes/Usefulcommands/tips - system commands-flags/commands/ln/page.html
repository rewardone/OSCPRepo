ln creates a link to file TARGET with the name LINKNAME. If LINKNAME is omitted, a link to TARGET is created in the current dirctory, using the name of TARGET as the LINKNAME

ln creates hard links by default, or symbolic links if the -s (--symbolic) option is specified. When creating hard links, each TARGET must exist.

good read/refresher on hard vs soft links: linuxgazette.net/105/pitcher.html
