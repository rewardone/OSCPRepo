The tiny utility fixelf removes the PF_X flag from the ELF program headers:

/* fixelf.c */
...
        Elf32_Ehdr *eh;
        Elf32_Phdr *ph;
...
        eh = (Elf32_Ehdr *) program;
...
        for (i=0; i<eh->e_phnum; i++) {
                ph = (Elf32_Phdr *)(program + eh->e_phoff + i*eh->e_phentsize);
                ph->p_flags &= ~PF_X;
        }
...
Today (2.6.21) this still works.
